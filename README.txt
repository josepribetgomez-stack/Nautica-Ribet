ACADEMIA NÁUTICA JOSEP RIBET — PROTOTIPO WEB

Cómo abrirlo
1. Descomprime la carpeta.
2. Abre index.html con Chrome, Safari, Firefox o Edge.
3. También puede subirse a cualquier alojamiento estático para hacer una prueba.

Qué incluye
- Página de inicio responsive.
- Página de venta del Curso PER.
- Test diagnóstico de 20 preguntas, funcional en el navegador.
- Página de contacto con formulario de demostración.
- Aviso legal, privacidad, cookies y contratación como borradores.
- Banner de cookies demostrativo.

Qué falta antes de vender
- Elegir y registrar dominio y marca.
- Incorporar fotografía real y datos profesionales definitivos.
- Conectar WordPress/Tutor LMS o adaptar esta web a la plataforma elegida.
- Conectar pagos mediante Stripe/WooCommerce.
- Conectar formularios a correo o CRM.
- Configurar alojamiento, SSL y copias de seguridad.
- Revisar legal y fiscalmente todos los textos.
- Crear vídeos, apuntes, banco de preguntas y aula del alumno.
- Revisar las preguntas del test antes de su uso comercial.

Archivos principales
- index.html: portada.
- curso-per.html: página de venta.
- test.html: test gratuito.
- contacto.html: contacto.
- styles.css: diseño visual.
- script.js: menú, FAQ, cookies, formularios y test.
